# GREEN-API Тестовое задание

## Описание проекта

Веб-приложение для тестирования методов GREEN-API, разработанное в рамках тестового задания на должность "Девопс разработчик".

## Функциональность

Приложение предоставляет интерфейс для работы с методами GREEN-API:

- **getSettings** - получение настроек инстанса
- **getStateInstance** - получение состояния инстанса
- **sendMessage** - отправка текстового сообщения
- **sendFileByUrl** - отправка файла по URL

## Как использовать

1. **Получение учетных данных:**
   - Зарегистрируйтесь в [личном кабинете GREEN-API](https://green-api.com/)
   - Создайте новый инстанс на бесплатном аккаунте разработчика
   - Отсканируйте QR-код для подключения вашего номера телефона
   - Получите `idInstance` и `ApiTokenInstance`

2. **Использование приложения:**
   - Откройте `index.html` в браузере
   - Введите `idInstance` и `ApiTokenInstance` в соответствующие поля
   - Используйте кнопки методов API для взаимодействия с GREEN-API
   - Результаты выполнения методов отображаются в поле "Ответ API"

3. **Отправка сообщений:**
   - Введите номер телефона (с кодом страны, без +)
   - Введите текст сообщения
   - Нажмите кнопку "sendMessage"

4. **Отправка файлов:**
   - Введите URL файла
   - Введите номер телефона получателя
   - Нажмите кнопку "sendFileByUrl"

## Требования к формату данных

- **Номер телефона:** 10-15 цифр без пробелов и специальных символов (например: 79123456789)
- **URL файла:** Полный URL к файлу (например: https://example.com/image.jpg)
